# My Webpage

Welcome to the personal webpage of Adil Nawaz, Ph.D. This website showcases my research, publications, and professional background in the field of privacy-preserving machine learning, federated learning, and Bayesian optimization.

## Project Structure

The project is organized as follows:

```
mywebpage
├── index.html            # Main entry point for the website
├── publications.bib      # Bibliographic information for publications
├── package.json          # npm configuration file
├── README.md             # Project documentation
├── resume.pdf            # Downloadable resume
├── styles                # Stylesheets for the website
│   ├── site.css          # Main styles
│   ├── themes.css        # Theme definitions
│   └── components        # Component-specific styles
│       └── _variables.css # CSS variables for design tokens
├── scripts               # JavaScript files for functionality
│   ├── publications.js    # Script for handling publications
│   ├── ui.js             # Script for UI interactions
│   └── interactions.js    # Script for adding interactivity
├── components            # Reusable HTML components
│   ├── header.html       # Header structure
│   ├── footer.html       # Footer structure
│   └── pub-item.html     # Structure for individual publication items
├── data                  # Data files
│   └── publications.json  # Structured publication data
├── assets                # Assets for the website
│   ├── fonts             # Font files
│   │   └── Inter.woff2   # Inter typeface
│   └── svgs              # SVG icons
│       └── icons.svg     # Icons used in the website
└── images                # Image files for visual content
```

## Features

- **Dynamic Publications**: The website dynamically loads and displays publications from a BibTeX file, allowing for easy updates and management.
- **Interactive UI**: Enhanced user experience with interactive elements, including toggles for visibility and animations.
- **Responsive Design**: The layout adapts to different screen sizes, ensuring accessibility on various devices.
- **Custom Themes**: Users can switch between different themes for a personalized experience.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   cd mywebpage
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Open `index.html` in your web browser to view the website.

## Usage

Feel free to explore the different sections of the website, including my research interests, publications, and contact information. You can also download my resume for more detailed information about my academic and professional background.
